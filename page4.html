Board Game: Fingers Crossed

Fingers Crossed is a board game based upon deception. Each player has a finger assigned to them. Each finger has different abilities. The point of the game is to lie to the other players in order to progress further in the game.

Aframe

Another project I worked on was rendering a simple 2-D picture into a 3-D version using Aframe. The original picture is Mondrain's Composition N0. 1 Lozenge with Four Lines. I also added some animation to make the blocks spin, which is not represented in the picture.
