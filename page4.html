Board Game: Fingers Crossed

My group and I created this game for a board game assingment where we were tasked with creating an original board game protoype of our choice. The result of this assignment was Fingers Crossed. Fingers Crossed is a board game based upon deception. Each player has a finger assigned to them. Each finger has different abilities. The point of the game is to lie to the other players in order to progress further in the game.

Roller Rush

This assignment invovled creating a game that could be run on a website using Javascript. The main code library we used for this project was Three.js The main focus of the game was to create a quick, fun, and repeatable mini-game. By having rounds that last at most 30 seconds and a limited number of rounds, the game feels quick and endlessly fun to play.
